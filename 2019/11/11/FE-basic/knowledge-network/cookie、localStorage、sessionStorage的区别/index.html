cookie、localStorage、sessionStorage的区别
2019-11-11|前端基础网络类问题

背景

webStorage

webstorage是本地存储，存储在客户端，包括localStorage和sessionStorage。

三者区别概括

应用场景

cookie

  • cookie每次请求都会自动带上，尽量精简，可以用于判断用户是否登录（cookie存放sessionId，是session在客户端的一种实现方式）

localStorage

  • 可以接替Cookie，实现购物车管理的工作；
  • HTML5游戏产生的一些本地数据；

sessionStorage

  • 表单内容多，优化用户体验，将表单页面拆分成多个子页面，按步骤填写，使用sessionStorage
